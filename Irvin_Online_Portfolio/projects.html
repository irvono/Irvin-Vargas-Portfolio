Unami Web Application

For my last required C# and Web Development class at Lane Community College I took the Capstone class. Here students would be paired into a team of 3 to 4 students. We would have to look for a client and develop a web application for them and present it by the end of the 12 weeks. The major requirements were to use ASP.NET with MVC5, implement Identity framework, and use GitHub as the process for sharing code between our team.
